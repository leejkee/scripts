## English

### Features

- Sync files from `ResourceDir` to `DestinationDir`.
- Default `ResourceDir` is `~\Pictures\icon`.
- Default `DestinationDir` is the current directory.
- Use `-h` or `--help` to display help information.


### Basic Usage

#### Linux
```bash
python main.py
```

#### Windows

```ps1
python.exe .\main.py
```
